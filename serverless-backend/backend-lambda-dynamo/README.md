## Hello, welcome to this learning repository! 👋


## Install

```sh
yarn install 
```

## How to use

1. Clone this repository
2. Go to methods.ts, you will see an blank function
3. Try out this function
4. Run tsc
5. Zip the generated file
6. Upload to Lambda in AWS and test
